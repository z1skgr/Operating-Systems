Detailed Description

A struct containing process-related information for a non-free pid.

This structure is returned by information streams.

int procinfo::argl

Argument length of main task.

Note that this is the real argument length, not just the length of the args field, which is limited at PROCINFO_MAX_ARGS_SIZE.

char procinfo::args[PROCINFO_MAX_ARGS_SIZE]

The first PROCINFO_MAX_ARGS_SIZE bytes of the argument of the main task.

If the task's argument is longer (as designated by the argl field), the bytes contained in this field are just the prefix.
